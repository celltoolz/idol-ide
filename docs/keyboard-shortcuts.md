# Keyboard Shortcuts

## File & Tabs

| Action | Shortcut |
|---|---|
| New tab | Ctrl+N |
| Open file | Ctrl+O |
| Save | Ctrl+S |
| Save As | Ctrl+Shift+S |
| Close tab | Ctrl+W |
| Exit | Ctrl+Q |

## Editing

| Action | Shortcut |
|---|---|
| Undo | Ctrl+Z |
| Redo | Ctrl+Y |
| Select All | Ctrl+A |
| Find & Replace | Ctrl+F |
| Add / remove cursor | Alt+Click |
| Clear all cursors | Escape / Click |
| Extend selection (per cursor) | Shift+Arrow |
| Toggle overwrite mode | Insert |
| Toggle comment | Ctrl+/ |
| Move line up / down | Alt+Up / Alt+Down |
| Duplicate line down (cursor follows) | Shift+Alt+Down |
| Duplicate line up (cursor stays) | Shift+Alt+Up |
| Run line / selection | Right-click menu |

## Navigation

| Action | Shortcut |
|---|---|
| Command palette | Ctrl+Shift+P |
| Go to Definition | F12 |
| Split editor | Ctrl+\\ |
| Toggle scroll sync | Scroll Lock |
| Change font | Ctrl+L |

## Debugger

| Action | Shortcut |
|---|---|
| Debug file / Continue | F5 |
